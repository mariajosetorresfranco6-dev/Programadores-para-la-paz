Durante este curso aprendí que la tecnología puede ayudar a fortalecer la participación ciudadana y la organización comunitaria. Comprendí la importancia de crear aplicaciones que faciliten la comunicación entre las personas. También aprendí cómo funcionan las bases de datos, el backend y las peticiones entre cliente y servidor. Entendí que la automatización permite ahorrar tiempo y mejorar los procesos. Además, conocí la importancia de proteger los datos personales de los usuarios. Aprendí a trabajar con Git y GitHub para controlar versiones de mis proyectos. Todo este conocimiento me permitirá desarrollar soluciones útiles para mi comunidad. Fue un curso muy valioso para fortalecer mis habilidades en desarrollo web.
