Sistema operativo:
Es el software principal del computador. Administra los recursos del sistema y permite que las aplicaciones funcionen correctamente.
Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.
Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y las respuestas que recibirán los usuarios.
Puerto:
Es un canal de comunicación que utiliza el servidor para recibir solicitudes. En esta actividad se utiliza el puerto 3000.
Logs:
Son registros que almacenan información sobre la actividad del servidor. Ayudan a monitorear el funcionamiento y detectar errores.
